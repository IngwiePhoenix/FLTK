class Fl_Clock

Class Hierarchy

   |
   +----Fl_Clock

Include Files

    #include <FL/Fl_Clock.H>

Description

This widget provides a round analog clock display and is provided for Forms compatibility. It installs a 1-second timeout callback using Fl::add_timeout().
